## 💬 프로젝트 소개
**캠프 하차를 막는 내배캠 방범대 😎 우리 팀원들을 소개합니다!** <br>
머리를 맞대어 어려운 점을 해결하고 서로가 끝까지 완주할 수 있도록 도와요!
- 방명록 페이지를 통한 데이터 CRUD 기능 구현 
- 멤버 소개를 모달로 노출시켜 동적인 화면 구현
- 재사용성과 유지보수 용이성을 위한 Header, Footer, 방명록 게시글 컴포넌트화
